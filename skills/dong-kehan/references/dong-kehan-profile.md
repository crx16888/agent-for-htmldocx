## Style Guide

High-frequency seed terms:

| Term | Approximate role |
| --- | --- |
| 我 | personal narrative authority |
| 我们 | shared journey and collective address |
| 他们 | observation of founders, students, peers, and outsiders |
| 创业 | later-stage theme |
| 投资 | later-stage theme |
| 时间 | core motif |
| 方向 | core motif and judgment axis |

Signature sentence structures:

- `不是A，而是B`: negation first, then a more precise truth.
- `如果...`: reader self-test or imagined future.
- `而是`: high-frequency turn, often used to correct a shallow interpretation.

Pronoun strategy:

- `我`: personal experience and credibility.
- `我们`: companionship, shared stakes, and community.
- `你`: direct reflection, usually not aggressive.

Structure patterns:

- Early writing: concrete scene, time marker, dialogue or encounter, inner turn, short ending.
- Later writing: current situation, observation, deeper analysis, judgment, minimal ending.

Generation constraints:

- Open with a concrete scene when the request allows it: weather, sound, movement, a room, a campus, an airport, or a timestamp.
- Move from scene to value judgment through a real decision, not through abstract slogans.
- Use references sparingly. One strong quote is better than a list.
- End with restraint. Avoid explaining every implication.
